Отладочная плата с микроконтроллером STM32F103ZET6 и TFT-дисплеем 2.8

On-board resources:
* CPU: STM32F103ZET6; (LQFP64, 128 KB flash, 20 KB SRAM, 12-битных АЦП; PWM, CAN, USB and other resources)
* With a 2.8-inch TFT true color touch screen module, color touch module configuration ADS7843 (or RSM1843 instead) controller, support for a SD card (SPI mode) can be used to store pictures, in support of AT45DBxxx of a DATA FLASH (can be used for storage)
* USB interface, make USB experiment
* RS232 interface (ISP download), make RS232 communicate experiment
* AD adjuster
* standard ARM JTAG 20 pin
* A RTC in support, the four users LED lights, a power indicator light, a USB communication indicator, two user keys
* Два кварцевых генератора: 8 МГц для микроконтроллера, 32.768 КГц для часов
* Питание через USB-порт
* Все выводы микроконтроллера разведены на гребенки контактов с шагом 2.54 мм
* Размер платы: 82 X 55 мм
* Пользовательские светодиоды:
LD3 - LED1 - PB0
LD4 - LED2 - PB1
(LD1 - LED)
(LD2 - PWR/5V)
* Пользовательские кнопки:
S3 - KEYA - PC13
S4 - KEYB - PB2
(S1 - BOOT0)
(S2 - RST)
